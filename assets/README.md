# Assets

Thả file `logo.png` (~96x96 px, hình vuông) vào folder này để hiện avatar KISS English ở góc dưới-phải template.

Nếu không có file, template tự fallback sang ô tròn màu vàng (#F7B500) làm placeholder — vẫn render bình thường, chỉ thiếu avatar.

Cách lấy avatar: vào Facebook Page KISS English, click avatar → lưu ảnh → đặt tên `logo.png`, drop vào đây trước khi build Docker.
